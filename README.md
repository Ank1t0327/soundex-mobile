# Soundex Mobile - PWA in JSX

A progressive web app (PWA) version of Soundex built with React and JSX, optimized for mobile devices.

## Features

✅ **Search Movies & TV Shows** - Real-time search using IMDb API  
✅ **Stream Content** - Direct links via playimdb.com  
✅ **Trailers** - Search on YouTube  
✅ **Watchlist** - Save titles for later  
✅ **History** - Track searches (max 20)  
✅ **Mobile First** - Touch-optimized UI  
✅ **Installable** - Add to home screen (iOS/Android)  

## Quick Start

```bash
npm install
npm start
```

Opens at [http://localhost:3000](http://localhost:3000)

## Build for Production

```bash
npm run build
```

## Installation as PWA

**Android**: Menu → Install app  
**iOS**: Share → Add to Home Screen
